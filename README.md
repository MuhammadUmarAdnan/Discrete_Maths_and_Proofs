# How to prove it: A structured Approach Solutions
This folder contains my solutions for the the book How to prove it by Velleman.

### About PDF
It contains images of work in order but is not very nicely formatted or colorful though understandale. A new section starting is written on top with same pen used for writing actual work and a new question is just written on left side followed by a parenthesis.

### A Note on solution manual previously avaialable
There are solution manuals that you can find on paid websites and one solution manual is also available for free if you search a bit but those contain final answers (lines of proof writing) while these solutions contain the scratch work and occasionaly some intuitions might be written along the way. So these are more like the process while the other is the final answer. Its recommended you use both to verify for a question you are stuck on.

### Important Note
It is important to note that while every effort has been made to ensure accuracy, there is no guarantee that every solution provided is entirely free of errors. As these are human-generated, mistakes can occasionally occur despite careful work. For more complex or difficult problems, the final answers have been cross-referenced and verified against a solution manual to maintain a high standard of correctness. However, please remain mindful that a slip in logic or calculation remains possible in any solution.

### Scratch work vs Thought process
Since my intention was to show the all the steps written, all images are as it is. I have cut nothing from in between even the mistakes that are cut later so as much intuitions can be grasped as possible. However, these don't show the thought process. A thought process is to occur in mind with a paper or board on which we keep repeating and correcting untill the logic clicks in mind but these can certainly help you reach the final intuition.
